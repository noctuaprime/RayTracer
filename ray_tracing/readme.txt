Instructions to compile:
type "make" into the command line

Instructions to run:
type "make run" into the command line

The main libraries that this program requires are GLFW3 and stb_image.

This program is designed to dislay two red planes and a metallic sphere between them. 
Input keys have been provided to allow the user to move the camera and move the main light used to light up the area.
